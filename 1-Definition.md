**jQuery** is a JavaScript library that makes it easier to:
-	Select HTML elements 
-	Modify the DOM 
-	Handle events 
-	Create animations 
-	Perform AJAX requests

## In short:
jQuery makes JavaScript shorter and easier.

## Why jQuery Was Created
Before jQuery, developers had to write long and complex JavaScript code.

## Example:
Without jQuery
```bash
document.getElementById("title").style.color = "red";
```
With jQuery
```bash
$("#title").css("color", "red");
```
Much shorter and cleaner.

## Does jQuery Still Matter in 2026?

Modern frameworks like:
-	React 
-	Vue 
-	Angular 
replaced jQuery in many projects.

But jQuery is still used in:
-	legacy enterprise systems 
-	WordPress themes/plugins 
-	internal dashboards 
-	quick prototypes 
Knowing jQuery helps you understand DOM manipulation deeply.

## How jQuery Works
jQuery works using the $() function.

### Example:
```bash
$("#title")
```
Then you can manipulate it.

### Example:
```bash
$("#title").hide();
```

### Meaning:
Select the element and hide it.
