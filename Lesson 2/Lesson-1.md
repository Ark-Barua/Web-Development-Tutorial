## Advanced Selectors

Now we move to powerful selectors used in real projects.

### You will learn:

- Attribute selectors
- First / Last selectors
- Even / Odd selectors
- Child selectors
- Not selectors
- Contains selectors
- Form selectors

These allow you to select very specific elements.

### Example:
```bash
$("p:first")
```
Selects the first paragraph.

Example:
```bash
$("p:last")
```
Selects the last paragraph.

Example:
```bash
$("p:even")
```
Selects even indexed paragraphs.
